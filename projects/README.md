# Projects

This folder contains capstone and project assignments for the CVI PKI Career Pathway cohort. Each project has its own subfolder with a README, template, and any supporting materials.

Projects build on the weekly labs and reflections completed in your portfolio repository. Before starting any project, make sure your weekly reflection files are up to date — project submissions should extend that work, not duplicate it.

## What's in Here

| Folder | Project | Phase |
|--------|---------|-------|
| `phase 1-reflection-project/` | Phase 1 Reflection Project | Phase 1 — Foundations |
| `pki-troubleshooting-runbook/` | PKI Troubleshooting Runbook | Phase 1 — Foundations |

More projects will be added as the cohort progresses through Phase 2 and beyond.

## How This Folder is Organized

Each project subfolder contains:
- `README.md` — what the project is, what's required, and where to submit
- A template file to complete and commit to your portfolio repository

Your actual submissions live in your **GitHub portfolio repository**, not here. Follow the file paths specified in each project README.
